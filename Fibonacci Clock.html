How to read this clock!

Realize that each square is worth a value. (hint, that where the Fibonacci comes into play!)
Small: 1, Medium: 2, Large: 3, Huge: 5

Each color represents either the current hour, or current minute, or both!
Red: hour, Green: minutes, Blue: hours and minutes

To find the current hour, simply add up each of the blocks with the colors RED or BLUE
To find the current minute, add up each of the blocks with the colors GREEN or BLUE, then multiply by 5
